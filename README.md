# Algorithms
Made in CLion, has a bunch of algorithms (located in Sorteermethode.h):

* Insertion Sort
* Shell Sort
* Selection Sort
* Heap Sort
* Merge Sort
* Quick Sort
* Dual Pivot QuickSort
* Binair Zoeken
* Interpolerend Zoeken
* Count Sort
* Radix Exchange Sort (Unavailable)
* MSD Radix Sort
* Ternaire Radix Sort (Unavailable)
* LSD Radix Sort
* Bucket Sort
